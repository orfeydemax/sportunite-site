# Стили Hero Section

## Aggressive (Боевой)

**Когда использовать:** Спорт, единоборства, фитнес, экстрим

```css
:root {
  --bg-primary: #0a0a0a;
  --bg-secondary: #1a0000;
  --accent: #ff2020;
  --accent-glow: rgba(255, 32, 32, 0.4);
  --text-primary: #ffffff;
  --text-secondary: #cccccc;
}
```

**Элементы:**
- Диагональные линии (45°)
- Красное свечение на CTA
- Пульсирующие анимации
- Uppercase заголовки
- Тёмный фон с градиентом

---

## Minimal (Минимализм)

**Когда использовать:** SaaS, Tech, Стартапы

```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f8f9fa;
  --accent: #0066ff;
  --text-primary: #1a1a1a;
  --text-secondary: #666666;
}
```

**Элементы:**
- Много белого пространства
- Один акцентный цвет
- Тонкие линии
- Простая типографика
- Subtle hover-эффекты

---

## Corporate (Корпоративный)

**Когда использовать:** B2B, Финансы, Консалтинг

```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f0f4f8;
  --accent: #0052cc;
  --accent-secondary: #00875a;
  --text-primary: #172b4d;
  --text-secondary: #5e6c84;
}
```

**Элементы:**
- Строгая сетка
- Приглушённые цвета
- Профессиональные иконки
- Чёткая иерархия
- Trust-элементы (логотипы клиентов)

---

## Creative (Креативный)

**Когда использовать:** Дизайн-агентства, Маркетинг, Креативные индустрии

```css
:root {
  --bg-primary: #1a1a2e;
  --gradient-start: #667eea;
  --gradient-end: #764ba2;
  --accent: #f093fb;
  --text-primary: #ffffff;
}
```

**Элементы:**
- Градиентные фоны
- Нестандартная геометрия
- Анимированные blob-формы
- Яркие акценты
- Эксперименты с типографикой

---

## Luxury (Премиум)

**Когда использовать:** Премиум услуги, Недвижимость, Автомобили

```css
:root {
  --bg-primary: #0d0d0d;
  --bg-secondary: #1a1a1a;
  --accent: #c9a962;
  --accent-secondary: #d4af37;
  --text-primary: #ffffff;
  --text-secondary: #a0a0a0;
}
```

**Элементы:**
- Золотые/бронзовые акценты
- Элегантные serif-шрифты
- Плавные анимации
- Высококачественные изображения
- Минимум элементов, максимум пространства
